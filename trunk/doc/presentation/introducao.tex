\begin{frame}{Ortogonalidade}
	\begin{block}{Definição Matemática}
		Dois vetores $x$ e $y$ são ortogonais num espaço vetorial $V$ se o produto interno $\left\langle x,y \right\rangle$ é zero. Esta situação é descrita por $x \bot y$.
	\end{block}
	\pause
	\begin{block}{Definição Adotada}
		Estamos interessados no quanto os elementos de um conjunto contribuem com informações não redundantes para a solução de um problema. Considerando que seja possível medir esta contribuição, e chamá-la de \textbf{significância}, podemos definir \textbf{ortogonalidade} como a média das significâncias dos elementos do conjunto.
	\end{block}
\end{frame}

\subsection{Objetivos}
\begin{frame}{Objetivos}
Explorar o problema de classificação associativa considerando ortogonalidade entre padrões freqüentes com a intenção de:
	\begin{itemize}[<+-| alert@+>]
		\item Minimizar o número de padrões utilizados na geração das regras;
		\item Diminuir a redundância das regras geradas;
		\item Diminuir a ambigüidade das regras geradas;
		\item Aumentar a efetividade das classificações.
	\end{itemize}
\end{frame}
